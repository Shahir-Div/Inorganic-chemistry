# كيمياء لاعضوية 1 - موقع تدريب تفاعلي

موقع ويب تفاعلي (صفحة واحدة) لمساعدة طلاب مادة الكيمياء اللاعضوية 1 على مذاكرة أسئلة الاختيار من متعدد بأسلوب التكرار المتباعد (Spaced Repetition).

## المحتوى
- **122 سؤال اختيار من متعدد** مع شرح لكل سؤال:
  - 50 سؤالاً من دورات (امتحانات) الفصل الأول السابقة.
  - 72 سؤالاً إضافياً مبنياً على ملخص المادة (نظرية بور، الجدول الدوري، البلورات، الروابط، الاستقطاب، نظرية الحقل البلوري...).
- كل الكود (HTML + CSS + JavaScript + الأسئلة) موجود داخل ملف واحد فقط: `index.html`.

## المميزات
- سؤال واحد يظهر في كل مرة مع 4 خيارات.
- عند الإجابة: تلوين أخضر للجواب الصحيح، وأحمر للخاطئ، مع شرح ينبثق تحت السؤال.
- الأسئلة التي تُجاب بشكل خاطئ تُعاد تلقائياً بعد ~5 أسئلة، وتتكرر حتى تُجاب بشكل صحيح.
- دورة أسئلة لا نهائية (لا يوجد "خط نهاية" - يمكن التدرب باستمرار).
- تصميم عصري داكن، متجاوب بالكامل مع الهواتف المحمولة، باللغة العربية (RTL).
- رابط سريع لحساب انستقرام: [@shahirhawara](https://instagram.com/shahirhawara)

## طريقة الاستخدام محلياً
1. افتح ملف `index.html` مباشرة في أي متصفح (لا حاجة لخادم أو تنصيب أي شيء).

## النشر على GitHub Pages
1. ارفع هذا المستودع (Repository) إلى GitHub.
2. من إعدادات المستودع Settings → Pages.
3. اختر Branch: `main` والمجلد `/ (root)`.
4. احفظ، وستحصل على رابط مباشر (مثال: `https://username.github.io/repo-name/`) يمكن مشاركته مع أي طالب.

## التعديل / الإضافة
- كل الأسئلة موجودة داخل مصفوفة `QUESTIONS` في وسم `<script>` داخل `index.html`.
- لإضافة سؤال جديد، أضف عنصراً بنفس الصيغة:
```js
{q:"نص السؤال", opts:["خيار أ","خيار ب","خيار ج","خيار د"], a:0, exp:"نص الشرح"}
```
حيث `a` هو رقم فهرس (index) الجواب الصحيح ضمن `opts` (يبدأ من 0).
